CSS

Es un lenguaje utilizado para describir la presentación y el diseño de un documento escrito en HTML o XML. Mientras que HTML estructura el contenido de una página web (como títulos, párrafos, imágenes, etc.), CSS se encarga del aspecto visual: colores, fuentes, tamaños, márgenes, alineaciones, animaciones y mucho más. ¿Qué puede hacer CSS? Cambiar colores de texto y fondo Modificar el tamaño y tipo de fuente. Organizar el contenido en columnas o cuadrículas. Agregar márgenes y rellenos Hacer que una página se vea bien en distintos tamaños de pantalla (diseño responsivo). Crear animaciones y transiciones visuales.
